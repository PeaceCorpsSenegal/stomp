Descriptiontop

With jSocial you can easily place social share buttons under your articles and pages on your website. This script uses the jQuery framework and works completely client side.

How to install top

First of all extract all files. Upload the Script/jsocial.js file to your server and include it together with jQuery in your webpage head tag.

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.min.js"></script>
<script type="text/javascript" src="jsocial.js"></script>

This step is only needed if you want to use the share buttons without an underscore in the name (ex: facebook, twitter, ...). Upload the images folder that you find under the Script directory.

On the place you want to show the buttons you put this code:

<div id="socialbuttons"></div>

Now you call the plugin on the html element in wich you want to show the social share buttons. This code you place in the head tag of your page.

<script type="text/javascript">
jQuery(document).ready(function($) {
$("#socialbuttons").jsocial({
highlight: true, 
buttons: "technorati,delicious,reddit,facebook,twitter,stumbleupon", 
icon_format: 'images/big/%s.png',
blanktarget: true
});
}); </script>

Each social share button gets a CSS class .jsocial_button so you can change the margins between the buttons in your css stylesheet.

.jsocial_button {
margin-right: 5px;
}

Parameters top

  • buttons: This is a comma sepperated list of the button you want to show (make sure you don't have whitespaces in this list)
  • blanktarget: If true the link will be opened in a new window/tab.
  • highlight: If true the icons will get an effect when hover.
  • icon_format: The path where the script can find the images and the icons extensions. The %s in the string will be replace with the name of the social network. For example if you set this path "images/big/%s.png" and for facebook the script will look for a facebook.png image in the images directory.
  • icon_hover_format: This is the same as the icon_format (for example "images/big/%s_hover.png"). Only this path format will be used when an icon is in the hover state. If you don't want to use this the leave this setting out.
  • shorturl: Set this to true if you want to use shorturls in the share links were needed. For example these are used in the twitter share link. If you set this to false, the twitter share button will use the long url.

Add your own websites top

It's very easy to add custom websites to the jsocial script. To do this you open the Script/jsocial.js file and scroll to the bottom of the file. There you'll find lines like this:

... 
jformat['nujij']	= "http://nujij.nl/ jij.lynkx?t={TITLE}&u={URL}&b={DESCRIPTION}"; 
jformat['ekudos']	= "http://www.ekudos.nl/artikel/ nieuw?url={URL}&title={TITLE}&desc={DESCRIPTION}"; 
jformat['digg']	= "http://digg.com/ submit?phase=2&url={URL}&title={TITLE}";
...

As you can see it's very easy to add websites to the list. You can use the following variables in the url:

  • {TITLE}: The title of the page
  • {URL}: The url of the page
  • {SHORTURL}: Tinyurl if the parameter shorturl is used, otherwise it returns the
    regular url
  • {DESCRIPTION}: Page description
  • {KEYWORDS}: Page keywords

To add a website you just add a line to the end of this list like this:

jformat['misterwong']	= "http://www.mister-wong.com/add_url/ ?bm_url={URL}&bm_title={TITLE}&bm_comment=&bm_tags={KEYWORDS}";
jformat['buzz']	= "http://www.google.com/reader/ link?url={URL}&title={TITLE}&snippet={DESCRIPTION}&srcURL={URL}&srcTitle={TITLE}"; 
jformat['myspace']	= "http://www.myspace.com/Modules/PostTo/Pages/?u={URL}";
jformat['customsite']	= "http://customsite.be/ share?url={URL}&title={TITLE}&description={DESCRIPTION}";

Mail top

In the new version you can now also use the mail tag to add a mail link. You can easily edit the data that is used for the mailto by opening the jsocial.js file and scrolling to the bottom. There you can find something like this:

jformat['mail'] = "mailto:to@email.com?SUBJECT={TITLE}&BODY={DESCRIPTION}-{URL}";

New share icons top

A much requested feature was the implementation of the new share buttons for twitter, facebook, linkedin and G+. Using these is easier then ever.

<script type="text/javascript">
jQuery(document).ready(function($) { 
$("#example1").jsocial({
	highlight: false,
	buttons: "twitter_horizontal,facebook_horizontal,googleplus_horizontalmedium,linkedin_horizontal"
      	});
});
</script>

Below you find a list of different share buttons you can use for each website:

  • Twitter: twitter_horizontal, twitter_vertical, twitter_none
  • Facebook: facebook_horizontal, facebook_vertical
  • Google+: googleplus(horizontal, googleplus_vertical, googleplus_horizontalsmall, googleplus_horizontalmedium
  • LinkedIn: linkedin_horizontal, linkedin_vertical, none
